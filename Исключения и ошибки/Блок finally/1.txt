Нет, в Java конструкция try должна всегда быть сопоставлена либо с блоком catch, либо с блоком finally, либо с обеими. Попытка использовать try без catch и finally приведет к ошибке компиляции.

Блок try предназначен для обработки исключений, и компилятор Java требует указать, как вы хотите управлять исключениями, которые могут возникнуть в этом блоке.
