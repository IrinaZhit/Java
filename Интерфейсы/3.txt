protected interface Interface {
    // методы интерфейса
}

Интерфейсы не могут быть объявлены с модификатором protected вне класса. Если интерфейс объявлен в классе, то он будет доступен для подклассов этого класса и классов в том же пакете. 
Но такое использование считается неправильным и приведёт к ошибке компиляции, если интерфейс находится вне класса.
