Вывод:
static logic
logic (1) id= 0
logic (2) id= 1
ctor id= 100

static logic — статический блок выполняется один раз при загрузке класса.
logic (1) id= 0 — первый нестатический блок выполняется до инициализации полей, поэтому id по умолчанию равно 0.
logic (2) id= 1 — второй нестатический блок выполняется после инициализации поля id значением 1, указанным при объявлении.
ctor id= 100 — в конструкторе поле id присваивается значение 100, переданное в параметре, и затем это значение выводится.
