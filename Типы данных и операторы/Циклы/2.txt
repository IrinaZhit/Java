Пример с break:
int i = 1;
while (i <= 10) {
    if (i == 7) {
        break; // Останавливаем цикл, когда i равно 7
    }
    System.out.println("i = " + i);
    i++;
}

Цикл начинается с i = 1 и продолжается до тех пор, пока i <= 10.
Когда i достигает значения 7, срабатывает оператор break, и цикл немедленно завершает выполнение.

Пример с continue:
int i = 1;
while (i <= 10) {
    i++;
    if (i % 2 == 0) {
        continue; // Пропускаем итерацию для чётных чисел
    }
    System.out.println("Odd number: " + i);
}

Цикл увеличивает значение i на каждой итерации.
Когда i — чётное число, срабатывает оператор continue, и текущая итерация пропускается, не выполняя вывод.
